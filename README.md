# Pricing Table
This project focuses on creating a responsive pricing table using Flexbox principles. The pricing table includes three subscription categories—basic, standard, and premium—designed to help customers easily compare features across different tiers.

Key Concepts and Tools Learned:
1. Flexbox Layout: learnt to use Flexbox to create a layout that effectively centers items and facilitates dynamic adjustments based on screen size.
2. Responsive Design: The use of media queries changes the flex direction from row to column for smaller screens, ensuring the layout remains user-friendly on various devices.
3. Styling Elements: Various CSS properties are applied, including setting equal widths, maximum widths, and styles for titles, prices, and buttons, enhancing the overall aesthetic of the pricing cards.
4. Personalization: Experimented with various styles, fostering creativity and subjective design.

Project Goal and Learning Outcomes:
1. The main goal of the project was to develop a visually appealing and responsive pricing table. 
2. Applied Flexbox for layout management, created responsive designs using media queries, and enhanced their CSS styling skills, showcasing creativity in web design.
